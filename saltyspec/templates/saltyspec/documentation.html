How it works

SaltySpec use SALT GUI to generate Use Case Specification. SALT GUI that’s used has to contain:

  1. Procedure
  2. Title
  3. Alternative scenario

How to install

SaltySpec can be installed by cloning our github repository below:

https://github.com/AgileRE-2022/SaltySpec  

Or simply download the zip file from this link:

https://github.com/AgileRE-2022/SaltySpec/archive/refs/heads/main.zip
